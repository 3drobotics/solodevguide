Licenses

Root access to Solo grants you access to all components of the system. This page documents the licenses of various components in the system.

  • Linux Subsystem
    • Yocto definition layer — Proprietary (3DR)
    • Solo package definitions — Proprietary (3DR)
    • uboot — GPLv2
    • Linux kernel with runtime patches — GPLv2
  • Video Encoder — Proprietary (external party)
  • Gimbal firmware — Proprietary (3DR)
  • Controller firmware — Proprietary (3DR)
  • SoloLink Manager — Proprietary (3DR)
  • DroneKit (solo version) — BSD
  • pymavlink (solo version) — LGPL
  • MAVProxy (solo version) — GPL
  • Ardupilot (solo version) — GPL
  • STM32 Loader — GPL
